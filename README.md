# Smart Resume Screening System

An AI-powered Resume Screening and Skill Gap Analysis System built using Python, Flask, Scikit-Learn, HTML, CSS, and JavaScript.

## Features

* Resume PDF Upload
* Resume Text Extraction using PyMuPDF
* Skill Extraction Engine
* Resume-Job Description Matching
* ATS Score Calculation
* Skill Gap Analysis
* Personalized Recommendations
* Interactive Analytics Dashboard
* Skill Visualization Charts

## Tech Stack

* Python
* Flask
* Scikit-Learn
* PyMuPDF
* HTML
* CSS
* JavaScript
* Chart.js

## Project Workflow

1. Upload Resume (PDF)
2. Enter Job Description
3. Extract Resume Content
4. Identify Technical Skills
5. Compare Resume with Job Requirements
6. Generate ATS Match Score
7. Identify Missing Skills
8. Generate Recommendations
9. Display Analytics Dashboard
